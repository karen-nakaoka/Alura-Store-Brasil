# 📊 Análise de Desempenho das Lojas – Alura Store

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte de um challenge de Análise de Dados, com o objetivo de auxiliar o Senhor João na decisão estratégica de qual loja da rede **Alura Store** deve ser vendida para viabilizar um novo investimento.

A análise foi realizada utilizando **Python**, com foco em manipulação de dados e visualização gráfica.

---

## 🎯 Objetivo

Avaliar o desempenho das quatro lojas com base nos seguintes indicadores:

- 💰 Faturamento total 
- 📦 Categorias mais e menos vendidas  
- 🏷️ Produtos mais e menos vendidos  
- ⭐ Média de avaliação dos clientes  
- 🚚 Frete médio  

Com base nesses dados, foi elaborada uma recomendação final fundamentada em evidências quantitativas.

---

## 🛠️ Tecnologias Utilizadas

- Python  
- Pandas  
- Matplotlib  
- Google Colab  

---

## 📂 Estrutura da Análise

### 1️⃣ Importação e Exploração dos Dados
- Leitura dos arquivos CSV
- Verificação de tipos de dados
- Análise exploratória inicial

### 2️⃣ Análise de Faturamento
- Cálculo do faturamento total por loja
- Comparação por meio de gráfico de barras

### 3️⃣ Ticket Médio
- Cálculo do valor médio por venda
- Identificação de diferenças estratégicas entre as lojas

### 4️⃣ Categorias e Produtos
- Identificação das categorias mais e menos vendidas
- Análise dos produtos com maior e menor volume de vendas

### 5️⃣ Avaliação dos Clientes
- Cálculo da média de avaliação por loja
- Comparação da satisfação entre as unidades

### 6️⃣ Frete Médio
- Cálculo do frete médio por loja
- Avaliação da eficiência logística

---

## 📊 Principais Resultados

- A **Loja 1** apresentou o maior faturamento total.
- A **Loja 4** apresentou o menor faturamento e o menor ticket médio.
- A **Loja 3** obteve a melhor média de avaliação dos clientes.
- A **Loja 4** apresentou o menor frete médio.
- O mix de produtos é semelhante entre as quatro lojas.
